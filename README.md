La verdadera ingeniería no consiste en reinventar la rueda, sino en saber qué herramientas utilizar para construir el futuro. Como fundador del proyecto tecno-ambiental estudiantil **Lifeplants**, mi enfoque no es la programación manual tradicional. Soy un constructor tecnológico que orquesta inteligencia artificial, automatización y herramientas de vanguardia para desarrollar soluciones con impacto real.

### ⚙️ Centro de Operaciones

- 🚀 **Liderando:** La visión estratégica, expansión y desarrollo de herramientas para **Lifeplants** y el programa **SDEAM**.
- 🏗️ **Construyendo:** Ecosistemas tecnológicos apoyándome en la dirección de asistentes avanzados (incluyendo el chatbot **RAIZ**).
- 🧠 **Orquestando:** Flujos de trabajo y generación de código impulsados fuertemente por **Claude Code** y **OpenCode**.
- 🌱 **Bases Lógicas:** Cimentando mi capacidad analítica con **CS50** de Harvard, para dirigir y auditar mejor mis herramientas.
